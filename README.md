# Phoenix.homes modern animated landing

Static HTML/CSS/JS landing page. No build tools required.

## Direction

This version is visually separate from DashVegas:

- desert luxury real estate
- Arizona sunset tones
- subtle mountain / skyline / city-grid atmosphere
- premium serif typography
- calm, modern animation
- mobile performance mode

## Files

- `index.html`
- `style.css`
- `script.js`
- `assets/`

## Deploy with GitHub Pages

1. Replace the old Phoenix.homes files with these files.
2. Commit changes.
3. Wait for GitHub Pages to update.
4. Hard-refresh or test in incognito/private browsing.
